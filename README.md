# Autonomous Research Agent (LangChain)

## Objective
This project implements an AI agent that performs autonomous research on a given topic and generates a structured report.

## Features
- Web search using DuckDuckGo
- Knowledge retrieval using Wikipedia
- ReAct-based agent using LangChain
- Structured report generation

## Tools Used
- LangChain
- OpenAI GPT model
- DuckDuckGo Search
- Wikipedia API

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Add OpenAI API key in `.env`

3. Run:
   python research_agent.py
